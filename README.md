# PhishXray

PhishXray is a lightweight desktop GUI triage tool engineered using Python's native `tkinter` frame to process and evaluate user-reported suspicious emails. It isolates hidden email attributes, flags domain spoofing variations, extracts embedded hyper-links, and scores the cumulative security threat vector using an intuitive, easy-to-read dark theme interface.

## Core Visual Capabilities
* **Interactive Desktop UI:** Built entirely without web dependencies, servers, or complex setup installations.
* **Instant Risk Heuristic Evaluations:** Assigns a risk threat score based on cryptographic protocol mismatches.
* **Envelope Spoofing Checks:** Correlates the visible email sender field directly with low-level hidden routing headers.
* **Payload Extractions:** Pulls active link pathways inside multi-part message text fields instantly for review.
